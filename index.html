Google Drive Organizer

Reads your Drive, classifies files by topic, creates folders, and moves everything automatically.

1 Google Cloud setup — one time only

This tool needs a free Google Cloud client ID to access your Drive. Takes about 5 minutes:

  1. Go to console.cloud.google.com/apis/credentials
  2. Create a project (or pick an existing one)
  3. Click + Create Credentials → OAuth client ID
  4. Application type: Web application
  5. Under "Authorized JavaScript origins", add:
  6. Click Create — copy the Client ID (ends in .apps.googleusercontent.com)
  7. Also enable the Google Drive API at this link
  8. Under OAuth consent screen, add your own Google email as a test user
The client ID is not a secret — it's safe to paste here. It only works from this exact page origin.
